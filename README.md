# 🔐 Password Generator (JavaScript Project)

## 📌 Overview

This is a simple and efficient **Password Generator Web App** built using **HTML, CSS, and JavaScript**.
It allows users to generate strong, random passwords with customizable options, helping improve security in daily digital usage.

---

## 🚀 Features

* Generate strong and secure passwords
* Customizable password length
* Includes:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
  * Special characters (!@#$%^&*)
* One-click copy to clipboard
* Clean and responsive UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

1. User selects password length
2. Chooses character types
3. Clicks "Generate Password"
4. Password is created instantly

## 💻 Installation & Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/MARIYUM890/password-generator.git
   ```
2. Open the project folder
3. Run `index.html` in your browser
